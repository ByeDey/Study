# 연산자 (Operators)

## 개념 요약

**연산자**는 데이터를 처리하고 연산하기 위한 기호입니다.
Java에는 산술, 비교, 논리, 대입, 증감 등 다양한 연산자가 있습니다.

---

## 산술 연산자

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `+` | 더하기 | `5 + 3` | `8` |
| `-` | 빼기 | `5 - 3` | `2` |
| `*` | 곱하기 | `5 * 3` | `15` |
| `/` | 나누기 | `5 / 2` | `2` (정수 나눗셈) |
| `%` | 나머지 | `5 % 2` | `1` |

```java
int a = 10, b = 3;

System.out.println(a + b);  // 13
System.out.println(a / b);  // 3 (소수점 버림)
System.out.println(a % b);  // 1

// 실수 나눗셈
double result = (double) a / b;  // 3.3333...
```

`int / int`는 소수점을 버립니다. 실수 결과를 원하면 `(double)`로 형 변환해야 합니다.

---

## 비교 연산자

결과는 항상 `boolean` (true / false)입니다.

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `==` | 같다 | `5 == 5` | `true` |
| `!=` | 다르다 | `5 != 3` | `true` |
| `>` | 크다 | `5 > 3` | `true` |
| `<` | 작다 | `5 < 3` | `false` |
| `>=` | 크거나 같다 | `5 >= 5` | `true` |
| `<=` | 작거나 같다 | `3 <= 5` | `true` |

---

## 논리 연산자

| 연산자 | 의미 | 설명 |
|--------|------|------|
| `&&` | AND | 둘 다 true일 때만 true |
| `\|\|` | OR | 하나라도 true면 true |
| `!` | NOT | true → false, false → true |

```java
boolean a = true, b = false;

System.out.println(a && b);  // false
System.out.println(a || b);  // true
System.out.println(!a);      // false

int age = 20;
boolean isAdult = (age >= 18) && (age < 65);  // true
```

## 대입 연산자

| 연산자 | 예시 | 풀어쓰면 |
|--------|------|---------|
| `=` | `a = 5` | `a = 5` |
| `+=` | `a += 3` | `a = a + 3` |
| `-=` | `a -= 3` | `a = a - 3` |
| `*=` | `a *= 3` | `a = a * 3` |
| `/=` | `a /= 3` | `a = a / 3` |
| `%=` | `a %= 3` | `a = a % 3` |

---

## 증감 연산자

| 연산자 | 의미 | 설명 |
|--------|------|------|
| `++a` | 전위 증가 | 먼저 1 증가 후 사용 |
| `a++` | 후위 증가 | 먼저 사용 후 1 증가 |
| `--a` | 전위 감소 | 먼저 1 감소 후 사용 |
| `a--` | 후위 감소 | 먼저 사용 후 1 감소 |

```java
int a = 5;

System.out.println(++a);  // 6 (먼저 증가 후 출력)
System.out.println(a++);  // 6 (출력 후 증가)
System.out.println(a);    // 7
```

---

## 삼항 연산자

`if-else`를 한 줄로 줄이는 연산자입니다.

```
조건식 ? 참일 때 값 : 거짓일 때 값
```

```java
int score = 85;
String result = (score >= 60) ? "합격" : "불합격";
System.out.println(result);  // 합격
```

## 연산자 우선순위

| 우선순위 | 연산자 |
|---------|--------|
| 1 | `++`, `--` (전위), `!` |
| 2 | `*`, `/`, `%` |
| 3 | `+`, `-` |
| 4 | `<`, `>`, `<=`, `>=` |
| 5 | `==`, `!=` |
| 6 | `&&` |
| 7 | `\|\|` |
| 8 | `=`, `+=`, `-=` ... |

헷갈리면 **괄호 `()`** 를 써서 명확하게 표현하세요.

---

## 주의할 점

- `int / int`는 소수점이 버려진다 → 실수 결과를 원하면 형 변환 필요
- 문자열에 `+` 사용 시 문자열 연결이 된다 (`"5" + 3` → `"53"`)
- 전위(`++a`)와 후위(`a++`)의 실행 순서 차이를 주의
- `&&`는 앞이 `false`면 뒤를 평가하지 않는다 (단락 평가)

## 면접 단골 질문

**Q. `i++`와 `++i`의 차이는?**
`i++`는 현재 값을 사용한 후 증가, `++i`는 먼저 증가한 후 사용

**Q. `==`와 `equals()`의 차이는?**
`==`는 주소(참조) 비교, `equals()`는 값(내용) 비교
